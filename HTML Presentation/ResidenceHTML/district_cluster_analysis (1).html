Analysis Interpretation

These scatter plots visualize the clustering results from three different perspectives:

  • Plot 1 shows the relationship between population density and elderly population percentage
  • Plot 2 examines how population density relates to median income
  • Plot 3 explores the connection between elderly percentage and median income

Each colored cluster represents districts with similar characteristics across these dimensions. The clustering helps identify patterns and groupings that might not be apparent when examining each variable separately.
